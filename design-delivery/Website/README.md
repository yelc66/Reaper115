# Reaper115 Website Kit

## Brand

- Product name: Reaper115
- Tagline: Telegram 管理控制台
- Style: iOS 26 inspired Liquid Glass, translucent panels, soft blue depth, compact operational UI.

## Colors

| Token | Hex | Usage |
| --- | --- | --- |
| Primary | `#007AFF` | CTA, active navigation, key data |
| Cyan | `#5ACBFA` | Gradients, Telegram accent |
| Sky | `#B0D9FF` | Icon highlights |
| Mist | `#E5F2FF` | App icon and page background wash |
| White | `#FFFFFF` | Glass surfaces |
| Ink | `#1C1C1E` | Primary text and soft shadows |
| Slate | `#8E8E93` | Secondary text |

## Type

Use SF Pro Display on Apple platforms, falling back to Inter, system-ui, -apple-system, BlinkMacSystemFont, and Segoe UI.

## Web Implementation

- Glass panels: `backdrop-filter: blur(20px)` and `rgba(255,255,255,0.62)`.
- Button hover: subtle lift, brighter fill, stronger blue shadow.
- Button active: slight scale down and darker translucent overlay.
- Radius: 8px for dense UI cards, larger only for app icon/logo surfaces.
