# Reaper115 Design Delivery

This package contains the iOS 26 / Liquid Glass style assets for Reaper115.

## Contents

- `Logo/`: master SVG plus PNG exports at 1024, 512, 256, and 128.
- `Favicon/`: PNG favicon exports at 16, 32, and 180.
- `Icons/`: functional icon SVGs plus PNG exports at 24, 32, 64, and 128.
- `Website/`: web color, typography, glass, and interaction notes.
- `Mobile/`: mobile home, chat, feature entry, and tab bar notes.
- `StyleGuide.pdf`: concise visual specification for handoff.

## Naming

Icons use the `function_size` convention, for example `upload_24.svg`, `download_32.png`, and `security_128.png`.
